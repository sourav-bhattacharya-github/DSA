# Data Structures and Algorithms (DSA) Preparation

Welcome to my comprehensive Data Structures and Algorithms (DSA) repository! This repository is organized systematically to build problem-solving skills from the ground up.

## 📚 Table of Contents

### 1. Learn the Basics

- Basic Programming Constructs (C++ / Java / Python)
- Logic Building (Patterns)
- Time and Space Complexity
- Basic Math for DSA
- Basic Recursion
- Basic Hashing

### 2. Sorting Techniques

- **Basic:** Selection Sort, Bubble Sort, Insertion Sort
- **Advanced:** Merge Sort, Quick Sort

### 3. Arrays

- **Easy:** Largest element, Second largest, Check if sorted, Remove duplicates, Move zeros, etc.
- **Medium:** 2Sum, Majority Element, Kadane's Algorithm, Stock Buy/Sell, Rearrange elements, Next Permutation.
- **Hard:** 3Sum, 4Sum, Subarrays with XOR, Merge Overlapping Subintervals, Missing/Repeating numbers.

### 4. Binary Search

- Binary Search on 1D Arrays
- Binary Search on Answers (Min/Max, Allocation problems)
- Binary Search on 2D Arrays / Matrices

### 5. Strings

- Basic and Easy String Problems
- Medium String Problems
- Hard String Problems

### 6. Linked Lists

- **Singly Linked List:** Introduction, Insertion, Deletion, Search
- **Doubly Linked List:** Introduction, Insertion, Deletion, Reverse
- **Medium Problems:** Middle of LL, Reverse LL, Detect Loop, Palindrome, Odd Even LL
- **Hard Problems:** Reverse in groups of K, Rotate LL, Flattening a LL, Clone LL

### 7. Recursion and Backtracking

- Get a Strong Hold (Subsequences, Combination Sum, Subset Sum)
- Hard Problems (N-Queens, Sudoku Solver, M-Coloring, Rat in a Maze)

### 8. Bit Manipulation

- Learn the Basics
- Interview Problems (Two Odd Occurring, Power Set, etc.)
- Advanced Maths (Sieve of Eratosthenes, Prime Factorization)

### 9. Stacks and Queues

- Learning Basics & Implementations
- Prefix, Infix, Postfix Conversions
- Monotonic Stack/Queue Problems (Next Greater Element, Trapping Rainwater)
- Implementation Problems (LRU Cache, LFU Cache)

### 10. Sliding Window & Two Pointers

- Medium Problems (Longest Substring without repeats, Max Consecutive Ones)
- Hard Problems (Minimum Window Substring, Subarrays with K distinct integers)

### 11. Heaps / Priority Queues

- Learning Implementations & Min/Max Heap
- Medium Problems (Kth largest, Merge K sorted arrays)
- Hard Problems (Find Median from Data Stream)

### 12. Greedy Algorithms

- Easy / Medium (N meetings in one room, Jump Game, Minimum Coins)
- Hard (Job Sequencing, Fractional Knapsack, Insert Interval)

### 13. Binary Trees

- Traversals (Inorder, Preorder, Postorder - Recursive & Iterative)
- Medium Problems (Height, Diameter, Check Balanced, Identical Trees)
- Hard Problems (LCA, Maximum Path Sum, Serialize/Deserialize)

### 14. Binary Search Trees (BST)

- Concepts & Traversals
- Practice Problems (Search, Insert, Delete, Kth Smallest, Valid BST)

### 15. Graphs

- Learning Graph Basics (Representations, Connected Components)
- BFS and DFS Traversals
- Topo Sort and Problems
- Shortest Path Algorithms (Dijkstra's, Bellman-Ford, Floyd Warshall)
- Minimum Spanning Tree (Prim's, Kruskal's, Disjoint Set)

### 16. Dynamic Programming (DP)

- 1D DP (Climbing Stairs, Frog Jump)
- 2D/3D DP and DP on Grids
- DP on Subsequences
- DP on Strings (LCS, Edit Distance)
- DP on Stocks
- DP on Longest Increasing Subsequence (LIS)
- Matrix Chain Multiplication (MCM) Partition DP

### 17. Tries

- Implementation and Basic Operations
- Advanced Problems (Maximum XOR of two numbers in an array)
